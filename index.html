CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Declarations contain 2 important components; property and value
  • CSS stands for: cascading stle sheets
  • Css is used to define: colors, aesthetics, visual layout, and orginization
  • There are 3 ways to use css to style webpage: inline css, internal css style sheet, external css style sheet
